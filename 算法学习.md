##  动态规划

动态规划应用于最优子结构问题，最优子结构:局部最优解也是全局最优解。所以全局问题能够描述成分解的若干子问题来解决

* [338. Counting Bits][4]
    * [Three-Line Java Solution][5]
    * 可以分为 `f[i]=f[i/2]+(i&1)`，如果 `i` 末尾为1，加上；否则 `i` 右移不改变1的位数
* [322. Coin Change][6]
    * [[C++] O(n*amount) time O(amount) space DP solution][7]
    * 与顺序无关
* [123. Best Time to Buy and Sell Stock III][8]
    * [Is it Best Solution with O(n), O(1).][9]
* [264. Ugly Number II][10]   
    * [O(n) Java solution][11]
* [62. Unique Paths][12]
    * [Java DP solution with complexity O(n*m)][13] 
* [96. Unique Binary Search Trees][14]
    * [dp problem. 10+ lines with comments][15]
* [72. Edit Distance][16]
    * 编辑距离：通过插入、删除和替换操作使两个字符串相同
    * [Java DP solution - O(nm)][17]

* [32. Longest Valid Parentheses][18]
    * 可以用 stack 实现，遇到右括号则弹出，最后不符合规则的都留在 stack 中，再求 stack 中两个相邻括号的差值，就是符合规则的括号数
    * DP 算法:v[i] 表示有括号时，且有左括号则 v[i] = v[i-1] + 2 同时还需加上之前的 v[i] += v[i-v[i]]
    * [My O(n) solution using a stack][19]

##  链表
[链表算法][38]
* 判断链表是否存在环
    * 设置两个指针，`slow` 每次前进一步，`fast` 前进两部，如果存在环，两个指针必定相遇
* 如果链表有环，查找环的入口点
    * 使用快慢两个指针，则相遇的时候，`slow` 走了 `s` 步，`fast` 走了 `2s` 步，`s=a+x`所以 `a+x=(n-1)*r` `x` 为环内走的距离，所以，`slow` 回到 `head`，再同时单步前进，相遇点为 `a` 步，就是环入口点。

##  百度实习
[第一题][42]

题目要求:是否能把所有的字符串首尾相连接，只有首尾的两个位置可以与其他字符串不匹配，其他的所有字符串都能首尾相匹配，那么这些首尾应该都是偶数个，只有最后字符串的首尾两个位置可以可以单独出来不为偶数的个数小于2个
```python
while False:
    try:
        ss = raw_input().strip()
        if ss == None:
            break
        n = int(ss)
        d = {}
        for i in range(n):
            s = raw_input().strip()
            if d.has_key(s[0]):
                d[s[0]] = d[s[0]] + 1
            else:
                d[s[0]] = 1

            if d.has_key(s[-1]):
                d[s[-1]] = d[s[-1]] + 1
            else:
                d[s[-1]] = 1
        m = 0
        for key in d.keys():
            if d[key]&1 == 1:
                m = m + 1
        if m > 2:
            print "No"
        else:
            print "Yes"
    except:
        break
```
[第二题][43]

题目要求:模拟 LRU 算法，需要注意的是，当初次调入 cache 的时候，可能存在冲突，需要先判断是否在 cache 中存在
```python
import sys
while False:
    try:
        try:
            m = int(sys.stdin.readline().strip())
            n = int(sys.stdin.readline().strip())
        except:
            break

        arr = []
        for i in range(n):
            arr.append(int(sys.stdin.readline().strip()))

        cache = []
        t,ans = 0,0
        for i in arr:
            if i in cache:
                continue
            ans = ans + 1
            if len(cache) < m:
                cache.append(i)
            else:
                cache[t] = i
                t = (t + 1)%(int(m))

        print ans
    except:
        break
```
